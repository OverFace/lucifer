  • 회원 탈퇴는 어떻게 진행하나요?

  • [회원정보]
  • 아이디/비밀번호를 잊어버렸어요.
왼쪽 로그인 버튼 밑 아이디/비밀번호 찾기 버튼을 눌러 변경을 진행하시면 됩니다.
  • [게임이용]
  • 레벨업이 어려워요.
레벨업 방법에는 다양한 방법이 존재하며, 필드에서의 몬스터 사냥 이외에도
여러 컨텐츠를 이용해 쉽게 레벨을 올릴 수 있습니다.

1. Stage 몬스터 이용
2. Stage 보스 클리어
3. 메인퀘스트, 서브퀘스트
4. 필드 몬스터

이 처럼 다양한 이벤트를 통해서 많은 양의 경험치와 보상을 획득할 수 있습니다.
따라서 이벤트들을 적극적으로 이용하시면 레벨 상승이 보다 수월해집니다.
  • [게임이용]
  • 스테이지는 어떻게 깨나요?
스테이지는 총 5 스테이지까지 있으며,
스테이지 진행에는 서브퀘스트, 메인퀘스트를 차례차례 진행하여
스테이지 마지막 보스를 처치하시면 스테이지가 클리어됩니다.
  • [게임이용]
  • 퀘스트 진행은 어떻게 하나요?
퀘스트는 메인퀘스트, 서브퀘스트, 반복퀘스트가 있으며,
메인퀘스트는 게임 스토리와 관련된 퀘스트로 마을 촌장에게 받아 진행할 수 있습니다.
서브퀘스트는 스테이지 내부 스토리 진행시 받을 수 있는 보조 퀘스트입니다.
반복퀘스트는 레벨업을 위한 단순 반복 퀘스트입니다.
  • [게임이용]
  • 아이템은 어떻게 얻나요?
아이템을 얻기 위한 두 가지 방법이 있습니다.

하나, 몬스터를 잡는다.
  몬스터 처치시 드랍되는 아이템을 획득합니다.
둘, 퀘스트 완료 보상을 받는다.
  퀘스트 진행 후 해당 NPC에게 완료 보고를 하면 보상이 주어집니다.
  • [설치/실행]
  • 게임접속이 안됩니다.
업데이트 이후 또는 이전에 발생하지 않던 문제가 발생될 경우, 쿠키 삭제를 통하여 문제를 해결해주시기 바랍니다.
1. 사용 중인 브라우저의 쿠키를 삭제해주세요.
  - 각 브라우저의 쿠키 삭제 방법은 아래를 참고해주시기 바랍니다.
2. 알약, V3 등 백신 프로그램을 이용하여 [PC최적화],　불필요한 항목을 [정리시작] 해주세요.
3. 사용 중인 웹 브라우저를 모두 종료해주세요.
4. PC를 재부팅 후 게임에 재 접속해주세요.

■ 익스플로러 브라우저 해결 방법
1. 익스플로러 메뉴에서 [도구]-[인터넷 옵션]으로 이동해주세요.
2. [일반] 탭의 [검색 기록]에서 [삭제]를 클릭 하여 주세요.
3. 항목을 체크 후 [삭제]를 진행해주시기 바랍니다.

■ 구글 크롬 브라우저 해결 방법
1. 크롬 메뉴에서 [설정]으로 이동해주세요.
2. 하단의 [고급 설정 표시]를 클릭해주세요.
3. 개인정보 탭의 [인터넷 사용정보 삭제]를 클릭해주세요.
4. 항목을 체크 후 [삭제]를 진행해주시기 바랍니다.
  • [설치/실행]
  • 스마트폰으로 게임 접속이 가능한가요?
안타깝게도 LUCIFER는 스마트폰으로 게임 접속이 불가능합니다.
빠른 시일 내에 스마트폰으로도 즐기실 수 있도록 노력하겠습니다.
  • [설치/실행]
  • 로그인을 했는데 다시 로그인을 하라고 나와요.
컴퓨터의 보안설정이 높게 설정이 되어있는 경우, 로그인이 정상적으로 되지 않는 경우가 있습니다.

이 경우, 아래와 같이 우선 설정을 바꿔주신 후에 다시 로그인 해보시길 바라겠습니다.
  1. InternetExploer창(인터넷 창) → [도구] → [인터넷옵션] → [보안] → [설정]을 낮음
  2. InternetExploer창(인터넷 창) → [도구] → [인터넷옵션] → [개인정보] → [설정]을 낮음
  3. 설정을 완료하신 후 다시 로그인 해보시길 바랍니다.
  • [설치/실행]
  • 게임 제한 연령만 이용할 수 있나요?
LUCIFER는 게임등급위원회 심의 결과 청소년이용불가 등급을 받았으며 이로인하여 만 18세 미만은 게임 이용이 불가능합니다.
  • [설치/실행]
  • 게임 실행을 위해 클라이언트 설치가 필요한가요?
LUCIFER는 별도의 설치 필요없이 바로 즐길수 있는 웹게임으로
웹브라우저를 통해 어디서든 쉽게 접속할 수 있습니다.
  • [설치/실행]
  • 게임 접속 불가 현상 또는 업데이트 내용이 적용되어 있지 않습니다.
게임 접속이 정상적으로 진행 되지 않거나 업데이트 내용이 적용 되지 않을 경우 아래의 해결 방법을 참고해 주시기 바랍니다.

1. 이용중인 웹 브라우저의 쿠키 및 인터넷 임시 파일 삭제를 하시기 바랍니다.
2. 백신 프로그램을 이용하여 PC 최적화와 불필요한 항목을 정리하시기 바랍니다.
3. 이용중인 웹브라우저를 모두 종료 후 다시 진행해 보시기 바랍니다.
4. PC를 재부팅 한 뒤 게임에 접속하여 주세요.

위 방법으로도 적용이 되지 않을 경우 1:1 문의를 주시면 도움을 드리겠습니다.

메인 홈페이지->고객센터 에 문의 사항란을 빠짐없이 자세히 작성해 주시면
빠른 시일 내에 접수하여 처리해 드리겠습니다.
  • [기타]
  • 스크린샷은 어떻게 찍나요?
웹게임에서는 대체로 스크린샷 자동 저장 기능을 제공하지 않고 있습니다.
기본적인 캡쳐 기능을 이용하여 스크린샷 처럼 사용하실 수 있으니
아래 내용을 참고하여 주시기 바랍니다.

[스크린샷 저장 방법]
1. 키보드 우측 상단의 PrintScreen 키를 눌러 주세요.
2. 컴퓨터의 그림판을 실행시켜 주세요.
3. 그림판에 붙여넣기(Ctrl+V)를 해주세요.
4. 원하시는 위치에 저장하시면 스크린샷이 됩니다.
